# Multi-Tenant HR Management Portal

Built automated tenant onboarding system with dynamic database provisioning to support multi-tenant architecture.

Implemented best database practices by assigning isolated logins per tenant, following the principle of least privilege, ensuring secure access control.

Developed reusable infrastructure modules for OTP, encryption, and notifications to standardize
cross-project services.
